# TU05 저화질 영상 생성 (에이전트 5)

연결 문서: [[00_인덱스]] | [[TU04_영상화_프롬프트_작성]] | [[TU06_고화질_영상_생성]]

## 중앙 프로세스

- 받는 것: 영상화 프롬프트 3개 + 베이스 이미지
- 처리: 저화질 프리뷰 영상 3개 병렬 생성
- 내보내는 것: 저화질 프리뷰 영상 3개(URL)
- 상태: UI 표시 후 `유저 응답 대기 중`

## 왼쪽 API 데이터 흐름

- 사용 API: 빠른/저비용 영상 API
- 입력 데이터: [프롬프트 3개 + 베이스 이미지]
- 출력 데이터: 프리뷰 영상 3개(URL)
- 다음 전달: 선택된 [프롬프트 + 베이스 이미지] 1세트를 TU06으로 익스포트

## 오른쪽 인터페이스 사용자 입력

- 화면 출력: 최신 배치 저화질 프리뷰 영상 3개
- 입력 1(선택): 3개 중 1개 선택
- 입력 2(재생성): 새 프리뷰 배치 생성
- 입력 처리: 이전 영상 배치는 archived로 유지
