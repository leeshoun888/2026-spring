# TU03 고화질 이미지 생성 (에이전트 3)

연결 문서: [[00_인덱스]] | [[TU02_저화질_이미지_생성]] | [[TU04_영상화_프롬프트_작성]]

## 중앙 프로세스

- 받는 것: 확정된 이미지 프롬프트 1개
- 처리: 고화질 이미지 1개 생성
- 내보내는 것: 고화질 이미지 1개
- 상태: UI 표시 후 `승인/재생성 대기`

## 왼쪽 API 데이터 흐름

- 사용 API: 고화질 이미지 API
- 입력 데이터: 프롬프트 1개
- 출력 데이터: 고화질 이미지 1개
- 다음 전달: 승인된 고화질 이미지 + 컷 정보를 TU04로 인계

## 오른쪽 인터페이스 사용자 입력

- 화면 출력: 고화질 이미지 1개
- 입력 1(승인): 현재 단계 완료, 다음 단계로 전달
- 입력 2(재생성): 동일 프롬프트로 다시 생성
- 입력 처리: 이전 결과는 `inactive/archived` 보관
