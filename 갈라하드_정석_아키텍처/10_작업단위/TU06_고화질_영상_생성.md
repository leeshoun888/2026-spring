# TU06 고화질 영상 생성 (에이전트 6)

연결 문서: [[00_인덱스]] | [[TU05_저화질_영상_생성]]

## 중앙 프로세스

- 받는 것: 확정된 [영상 프롬프트 + 베이스 이미지] 1세트
- 처리: 고화질 영상 1개 생성
- 내보내는 것: 고화질 영상 1개
- 상태: UI 표시 후 `승인/재생성 대기`

## 왼쪽 API 데이터 흐름

- 사용 API: 고품질/고비용 영상 API
- 입력 데이터: [프롬프트 + 베이스 이미지] 1세트
- 출력 데이터: 고화질 영상 1개
- 다음 전달: 승인 시 최종 편집/합성 파이프라인으로 인계

## 오른쪽 인터페이스 사용자 입력

- 화면 출력: 고화질 영상 1개
- 입력 1(승인): 제작 완료, 후속 파이프라인으로 전달
- 입력 2(재생성): 동일 세트로 다시 렌더링
- 입력 처리: 이전 결과는 `inactive/archived` 보관
