# 용어와 공통 규칙

연결 문서: [[00_인덱스]] | [[02_전체_플로우_요약]]

## 용어

- [에이전트 인풋]: 다른 작업단위에서 넘어온 시스템 입력 묶음
- [유저 인풋]: UI에서 선택/재생성/승인으로 들어오는 입력
- 기본 테스크: 시스템 입력을 받으면 바로 실행되는 처리
- 인터액티브 테스크: 유저 입력이 들어오면 추가로 실행되는 처리
- 익스포트: 다음 작업단위로 결과를 넘기는 전달
- 임포트: 이전 작업단위에서 결과를 받는 수신
- 파이프라인 트리거: 이전 결과가 다음 기본 테스크를 자동 시작시키는 연결

## 공통 상태

- 생성 중
- 유저 응답 대기 중
- 승인/재생성 대기
- 완료

## 공통 보관 정책

- 선택되지 않은 결과는 삭제하지 않고 `inactive/archived`로 보관
- 재생성은 새 배치를 만들고, 이전 배치는 이력으로 유지
- UI는 기본적으로 최신 배치 중심으로 표시

## 공통 사용자 입력 처리

- 선택: 여러 후보 중 1개 확정
- 재생성: 동일 목적의 새 후보 배치 생성
- 승인: 현재 단계를 확정하고 다음 단계로 인계
